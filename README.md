# Node.js Random Emoji Generator

Generate random emojis on the command line

## Random emoji

```bash
$ node index.js
❤️
$ node index.js
🍎
```

## Emoji by name

```bash
$ node index.js strawberry
🍓
```

Incorrect emoji names will print an error

```bash
$ node index.js strawberryyyyyy
Invalid emoji name
```
